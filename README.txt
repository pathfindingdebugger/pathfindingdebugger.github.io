The pathfinding debugger project

The project pitch was to create an algorithmic debugger for the family of pathfinding algorithms.The debugger was to include a visual debugger so users could visually see the search being conducted.

About Us

As part of the units: FIT1041 and FIT2028 at Monash University, this is a research project supervised by Daniel Harabor and Michael Wybrow, and conducted by Jay John Wingate and Surayez Rahman
